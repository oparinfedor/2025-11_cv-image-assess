# 2025-11_cv-image-assess
Модуль для оценки качества изображений
Руководство к использованию:

1. Загрузите файлы image_quality_assessor.py и demo.py
2. В директорию с файлами сохраните изображение в формате .png или .jpg
3. В командной строке файла demo.py выполните команду python demo.py --image + название файла с форматом, например: python demo.py --image name.png
4. Результаты оценки изображения будут выведены после того, как вы закроете изображение.
